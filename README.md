# AIPhone

A release-ready macOS companion for `Open-AutoGLM`, written entirely in SwiftUI.

## Current UI
- hidden macOS window action buttons
- rounded prompt container inspired by ChatGPT
- send button
- no Python changes

## Run
```bash
cd aiphone
swift run
```

## Open in Xcode
Open `Package.swift` in Xcode and run the `AIPhone` target.
